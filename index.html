PM5 Monitor

This application connects to a Concept2 PM5 monitor over Bluetooth and displays all workout variables.

Requirements

  • Web Bluetooth supported browser
  • PM5 attached to a Concept2 ergometer

Usage

  • On the PM5, More Options → Turn Wireless ON
  • Click the Connect button above and pair to the PM5.
  • Wait a moment while the browser reads the device characteristics.
  • Click any field to highlight it.
  • Set up a workout on the PM5. Start rowing, skiing, or biking. Watch the numbers change.

Tested on Chrome on macOS. Source on GitHub.
